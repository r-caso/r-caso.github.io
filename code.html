Code & Frameworks

Frameworks

I am currently developing three C++ frameworks (not a Python fan, sorry!), one for modeling static semantics, another for dynamic semantics, and a third one for capturing expressive/not-at-issue content.

The frameworks are still experimental, I will be releasing them via GitHub after proper testing and code cleanup.

Applications

Also in the way are some apps to use those frameworks in a friendly manner (written in Qt6).

In the mean time, you can explore a bare bones evaluator for Dynamic Predicate Logic. It is still in beta, with a lot of room for improvement (and surely quite a few bugs). If you want to report a bug or request a feature, for the time being, please contact me.

If you have security worries about this web application, check the WebAssembly page. Briefly, the app itself runs in a sandbox with the same security as your browser.

If you have privacy worries, contact me and I'll provide temporary access to the source code of the project.
